# Supply-Chain-Performance-Analytics-Dashboard
Project Overview

This project analyzes supply chain data to uncover insights into sales performance, inventory management, supplier efficiency, manufacturing operations, logistics, and product quality. Using SQL for data cleaning and analysis and Power BI for visualization, the dashboard provides actionable insights that support data-driven business decisions.

Objectives
- Analyze revenue across products, suppliers, and customer segments.
- Monitor inventory levels and product availability.
- Evaluate supplier and manufacturing performance.
- Analyze shipping costs and transportation efficiency.
- Identify quality trends using defect rates and inspection results.

Tools & Technologies
- SQL – Data cleaning and exploratory data analysis (EDA)
- Power BI – Interactive dashboard creation
- Microsoft Excel – Initial data review
- Dataset – Supply Chain Dataset from Kaggle
